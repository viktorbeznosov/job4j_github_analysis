Приложениe для Анализа Данных с GitHub.
## Описание

Spring Boot приложение для анализа данных с GitHub API. Приложение автоматически собирает информацию о репозиториях и коммитах пользователя, сохраняет данные в PostgreSQL и предоставляет REST API для доступа к ним.

## 🚀 Функциональность

- **Сбор данных с GitHub API**
    - Получение списка репозиториев пользователя
    - Получение коммитов для каждого репозитория

- **Автоматизация по расписанию**
    - Периодический сбор данных с настраиваемым интервалом
    - Использование `@Scheduled` для выполнения задач

- **Асинхронная обработка**
    - Сохранение репозиториев асинхронно (`@Async`)
    - Сохранение коммитов асинхронно

- **REST API**
    - Получение всех репозиториев
    - Получение коммитов по имени репозитория
    - Создание нового репозитория

- **Миграции баз данных**
    - Управление схемой через Liquibase
    - Автоматическое создание таблиц

## 🛠 Технологии

- **Java 17**
- **Spring Boot 4.0.4**
- **Spring Security** + **JWT**
- **Spring Data JPA**
- **PostgreSQL / MySQL**
- **Liquibase / Flyway** (миграции базы данных)
- **Maven** (сборка проекта)
- **Swagger/OpenAPI** (документация)
- **JUnit 5, Mockito** (тестирование)
- **GitHub Actions** (CI/CD)
- **Checkstyle** (код-стайл)

## 📋 Требования

- Java 17 или выше
- Maven 40+
- PostgreSQL 18+
